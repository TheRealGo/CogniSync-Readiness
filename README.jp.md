# CogniSync-Readiness

[English](README.md) | [日本語](README.jp.md)

**CogniSync-Readiness** は、科学的に実証された2つのタスク「**精神運動覚醒検査 (PVT)**」と「**フランカー課題 (Flanker Task)**」を用いて、あなたの**認知的準備状態 (Cognitive Readiness)** を測定・記録するための専門的なツールです。

このプロジェクトの主な目的は、ユーザーが「集中できている」という主観的な感覚と、客観的なパフォーマンスデータを同期（Sync）させることを支援することです。基礎的な覚醒レベル（PVT）と実行機能における抑制制御（Flanker Task）を測定することで、脳の現在の状態を正確に可視化します。

## 主な機能
- **高精度測定**: 反応速度、ラプス（反応の遅れ）、および干渉制御（ノイズの無視能力）を高精度にトラッキングします。
- **データの同期と蓄積**: 結果をローカルデータベースに記録し、長期的なトレンドの分析や、パフォーマンスが最大化される時間帯の特定を可能にします。
- **自己キャリブレーション**: 主観的な自己評価と客観的な指標を比較することで、疲労や睡眠不足による認知の歪みや「メンタルな錯覚」を排除します。

高い精神的明晰さを必要とするすべての人にとって、**CogniSync-Readiness** は、脳が真に「準備完了」状態にあるときのみパフォーマンスを発揮できるようにするためのデジタルな「ガードレール」として機能します。
